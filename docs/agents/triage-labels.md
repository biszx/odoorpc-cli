# Triage labels

The triage skill expects a mapping for the five canonical roles. These are the default mappings for this repository (no overrides provided):

- `needs-triage` — maintainer needs to evaluate
- `needs-info` — waiting on reporter for more information
- `ready-for-agent` — fully specified; an AFK agent can pick this up
- `ready-for-human` — requires human implementation
- `wontfix` — will not be actioned

If your repository uses different label strings, edit the mappings below or update your GitHub labels to match these names.
